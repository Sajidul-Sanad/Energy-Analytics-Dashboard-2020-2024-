# Energy Analytics Dashboard (2020–2024)

This project contains a two-page Excel dashboard analyzing synthetic energy data for Halle (Saale), Germany.  
It includes KPIs, PivotTables, PivotCharts, and slicers to provide management and operational insights similar to a utility/energy provider.

## Features
- Total Energy, Cost, CO₂, Outages, Grid Loss, Customers
- Monthly Energy & Peak Load Trends
- Renewable Share vs Grid Loss (Yearly)
- Outages by Region (Pie)
- Energy & Cost by Region (Yearly)
- Energy & Cost by Customer Type
- Slicers for Year, Month, Region, Customer Type
